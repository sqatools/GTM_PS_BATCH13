- Roles and Responsibility (SDET )

  - Maintaining Existing code (fix locator issue, workflow changes, flaky test cases)
  - Write new automation code for new feature with existing framework.
     - add locator file
     - add page file
     - add data file
     - add test file with all possible scenarios.

  - Analyze the automation code execution and report issue, if any product bug is there.
    - find automation issue then fix it.
    - find product issue then raise a defect in defect management (JIRA)

  - Maintaining Automation Pipeline and its run.

  - Design new automation framework for any new project is introduced.
    - create automation folder structure
    - make sure framework follows the standard (POM)
    - framework should have proper logging then user can debug easily.
    - Make sure create maintainable pages for each feature.
